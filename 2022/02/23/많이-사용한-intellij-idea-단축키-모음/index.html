많이 사용한 IntelliJ IDEA 단축키 모음

내가 많이 사용한 IDEA 단축키 모음

찾기, 검색 관련

⌘ + E : 최근 실행했던 파일 확인 및 이동

⌘ + F : 위치한 파일에서 특정 문자, 문자열 찾기

⌘ + ⇧ + F : Project안 모든 파일에서 특정 문자, 문자열 찾기

⇧ + ⇧ : Search Everywhere(전체 검색), 모든 항목을 검색할 수 있는 검색 상자가 열립니다.

⌘ + R : 위치한 파일안에서 검색 내용과 일치하는 텍스트를 쉽게 일괄 변경할 수 있도록 지원

⌘ + ⇧ + R : Project안 모든 파일에서 검색한 텍스트와 일치하는 내용을 모두 변경

코딩 관련

⌃ + ⏎ : 생성. getter and setter 생성, 파일 생성 기능

⌘ + ⇧ + ⏎ : Complete Current Statement(현재 구문 완성) 세미콜론 추가, for 작성 중 이 단축키를 누르면 (), {}를 자동 생성

⌘ + ⇧ + (↑ or ↓) : 위치 또는 드래그한 라인들을 가지고 이동

⌥ + ⏎ : 오류가 발견된 경우 해당 오류 위에 커서를 놓고 누르면 문제에 대한 수정 제안 목록이 표시
